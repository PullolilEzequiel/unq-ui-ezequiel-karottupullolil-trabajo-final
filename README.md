# Trabajo final Interfaces de usuario - UNQ 



## Descargar e instalar proyecto

1. Descargar codigo fuente
```bash
git clone && cd ./unq-ui-ezequiel-karottupullolil-trabajo-final
``` 
2. Instalar dependencias y iniciar proyecto 
```bash
npm run install && npm run start 
``` 

3. Ir a la pagina `http://localhost:5173/` 

4. Jugar
